# `src/services`

Lógica de acceso a datos y utilidades de lado cliente.

- `doctrineService.ts` — módulos y lecciones (semilla; Firestore opcional más adelante).
- `ttsService.ts`, `mediaService.ts` — voz y medios.
- `*.service.ts` — Firestore (notas, temas, progreso, perfiles).
- `localNotesQueue.ts` — cola offline de notas en `localStorage`.
